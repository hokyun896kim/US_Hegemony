미국 헤게모니 트리 v1.2

연간 YoY 옆에 분기 TTM(최근 4분기 연환산)을 나란히. 둘의 차이가 가속/둔화 신호입니다 — 분기가 연간보다 높으면 헤게모니가 막 붙기 시작한 것(형님의 "남보다 먼저"). 3층 구조: 1층 펀더(연간+분기) · 2층 보조(상대강도·갭·VIX) · 밸류에이션(PER) 체크. 차트로 고르지 않되, 차트 없이 사지 않는다. 종목을 펼치면 최신 8-K 실적 원문(보도자료·CFO 코멘터리)으로 바로가기 — 가이던스·톤은 점수화하지 않고 1차 자료를 직접 읽습니다.

v0.9. 순위 정밀화(2종목 사후검증 반영) — ① 분기TTM 기저효과 감지: 연간이 멀쩡해도 분기TTM 영업이익YoY가 폭등(흑자전환)하면 기저효과로 강등(EXPE형 차단). ② RS6M+RS3M 결합: 6M 양수인데 3M 음수면 '정점 통과·냉각' 감점(BKR형 차단), 6M 약세인데 3M 반등이면 '변곡' 가점 — 기존 RS6M 사각지대 해소. ③ PER 결측 무벌점 금지: 손익 오염 가능성으로 -6 + 검증 플래그. 결과적으로 EXPE·BKR 같은 '겉만 1등'은 ⚠검증 필요로 강등됩니다.
v0.8. 순위 계산 합리화 — TOP5 스코어가 ① 매출 동반 성장(고품질)을 핵심 가점으로, ② 흑자전환·기저효과·일회성(영익YoY +80%↑ 또는 가속 ±60p↑)을 자동 추정해 강하게 감점(가짜 1등 방지), ③ 비용절감형(매출 역성장)·고PER·피크아웃 감점, ④ 과도한 약세는 매복이 아닌 가치함정으로 절제. 각 후보에 품질 등급(고품질/보통/저품질/⚠️기저효과) 표기. 트레이드 카드도 동일 기준의 강세·약세 논거로 보강.
v0.7. ① 저장소 표준화 — 워치리스트·일지를 브라우저 기본 저장소(localStorage)로 저장해 어떤 호스팅(Netlify 등)에서도 유지됩니다. ② 밸류에이션 축 — 트레이드 카드에 PER/Forward PER 표시 + "헤게모니 강한데 비싼" 함정 경고. (PER 데이터는 tree.json에 pe/fpe 필드가 있을 때 표시)

이 도구로 무엇을 할 수 있나

이 도구는 미국 주식의 주도 산업과 그 안의 주포를 펀더멘털만으로 발견하기 위한 레이더입니다. 차트로 종목을 고르지 않습니다. 대신 SEC 공시(1차 자료)에서 뽑은 실적 숫자로 "지금 어느 산업에 이익 레버리지가 붙고 있는가"를 보여줍니다.

🎯 핵심 활용 4가지

① 주도 산업 발견 (탑다운)
11개 대섹터 → 79개 세부산업 순으로, 헤게모니가 강한 산업이 위로 정렬됩니다. "지금 미국에서 어디가 끌고 가는가"를 편견 없이 데이터로 확인.
② 주도 산업 안의 주포 식별
세부산업을 펼치면 구성 종목이 스프레드순으로 나옵니다. 같은 산업에서 누가 이익 레버리지를 끌고 있는지 비교.
③ 가속 종목 포착 (남보다 먼저)
'헤게모니 가속' 탭 — 최근 분기(TTM)가 연간 평균보다 강해진 종목. 이익 레버리지가 막 붙기 시작하는 초입 구간을 찾는 용도.
④ 1차 자료로 직접 점프
관심 종목의 최신 8-K 실적 원문(보도자료·CFO 코멘터리)으로 바로가기. 가이던스·경영진 톤은 여기서 직접 읽거나 다른 AI에 던져 분석.

📖 숫자 읽는 법 (가장 중요)

이 도구의 모든 판단은 헤게모니 스프레드 하나에서 시작합니다. 공식은 단순합니다:

=
헤게모니 스프레드 = 영업이익 증가율(YoY) − 매출 증가율(YoY)
매출이 10% 늘 때 이익이 30% 늘면 → 스프레드 +20p. 이익이 매출보다 빠르게 느는 것 = 가격결정력·규모의 경제·비용효율이 작동 중이라는 신호입니다.

실전 해석 4단계 (한 종목을 펼쳤을 때):

1단계 · 연간 스프레드를 본다
양수(+)면 일단 이익 레버리지가 있는 기업. +20p 이상이면 강한 헤게모니. 음수면 매출은 느는데 이익이 안 따라오는 마진 압박 상태 → 일단 제외 후보.
2단계 · 분기TTM과 비교한다 (피크아웃 체크)
연간이 +30p인데 분기TTM이 +10p로 낮아졌으면 → 헤게모니가 정점을 지나는 중(피크아웃 의심). 반대로 분기가 연간보다 높으면 → 최근 더 강해지는 중. 이 비교가 "지금 들어가도 되나"의 핵심입니다.
3단계 · 가속을 본다 (타이밍의 핵심)
가속이 양수(녹색) = 이익 레버리지가 막 붙기 시작하는 초입. 시장이 아직 모를 때 미리 발견하는 자리. 이게 형님이 가장 원하는 "남보다 먼저"입니다.
4단계 · 상대강도와 밸류로 타이밍을 가른다
펀더 좋은데 RS 약함 = 시장이 아직 안 깨움 → "매복 후보"(기다렸다 진입). 펀더+RS 둘 다 강함 = 이미 진행 중 → 추격 주의. 추가로 PER이 과도하게 높으면 헤게모니가 강해도 가격이 비싼 것 → 진입 신중.

💡 실전 시나리오 예시

📈 이상적인 매수 후보 (관찰 가치 높음)
연간 +15p / 분기TTM +25p / 가속 +10p / RS6M -5% / PER 18
→ 해석: 헤게모니가 가속 중인데 시장은 아직 안 깨웠고 밸류도 부담 없음. 전형적인 "매복" 자리. 8-K 원문으로 가이던스 확인 후 분할매수 검토.
⚠️ 피해야 할 함정 (피크아웃 + 고밸류)
연간 +40p / 분기TTM +8p / 가속 -32p / RS6M +60% / PER 70
→ 해석: 과거엔 폭발했지만 최근 둔화 중인데 주가만 이미 많이 올랐고 PER도 비쌈. 고점 추격 위험. 보류.
🔍 더 봐야 하는 애매한 경우
연간 +12p / 분기TTM 데이터없음 / RS3M +8%
→ 해석: 분기 데이터가 없어(은행·특수섹터 등) 판단 재료 부족. 트레이드 카드의 8-K 원문·뉴스 링크로 직접 확인 필요.

🤖 AI로 매매팁 얻는 법

숫자만으로 부족할 때, 종목 트레이드 카드 하단의 "분석 프롬프트 복사" 버튼을 누르면 이 종목의 모든 데이터가 채워진 질문이 복사됩니다. 그걸 GPT나 Claude에 붙여넣으면 — 산업 맥락, 피크아웃 여부, 리스크 3가지, 진입 체크리스트까지 정리해줍니다. 1차 자료(SEC·IR 링크)를 같이 던지면 분석 품질이 크게 올라갑니다.

⛔ 이 도구가 하지 않는 것

매수·매도 신호를 주지 않습니다. 목표가·밸류에이션 판단도 하지 않습니다. 차트 지표(이동평균·RSI 등)는 일절 없습니다. 이건 "어디를 깊게 팔지" 좁히는 발견 도구이지, 매매 결정 도구가 아닙니다. 최종 판단은 1차 자료를 직접 읽고 형님이 내립니다.

데이터 주의: 스냅샷입니다(실시간 아님). 분기TTM은 약 83% 종목만 확보(은행·일부 특수섹터 N/A). 가속 ±150p 초과는 전년이익≈0 기저효과라 회색 처리. 외국기업 ADR·일부는 SEC 분기 미보고로 빠질 수 있습니다.
